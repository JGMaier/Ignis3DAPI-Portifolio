# Ignis3DAPI-Portifolio
API em FastAPI responsável pela comunicação da aplicação Ignis3D com o banco de dados.
Ela fornece endpoints para registro e busca de credenciais, além de realizar a validação de acesso de usuários na aplicação.
Esse projeto demonstra boas práticas de autenticação e integração entre frontend e backend, utilizando FastAPI, SQLAlchemy e Pydantic.
